# FTXUI项目开发指南
### 次文本为我在使用FTXUI开发项目过程中的见闻

## 区别操作对象种类
- 总的来说，在使用FTXUI开发项目过程中，经常用到的是**Element**和**Component**等
- **Element**是被渲染的对象，对于任何一个要被渲染的Component类型的对象，都得先转换为 *Element*才可以进行装饰和渲染操作。显然，要对一个Component的对象进行管道操作的时候，及通过 | 标识符进行添加属性装饰的时候，先得 -> Render()，在进行管道操作
- **Component**是交互组件，可以理解为动图，需要转化为Element才可以进行渲染。因为该类型是根据用户的操作而改变的，并非一成不变的，所以在一些需要Component类型对象参数进行渲染的情况下，需要进行引用操作，以达到实时更新渲染的效果
